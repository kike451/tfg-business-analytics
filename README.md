# TFG - Business Analytics

## Predicción de abandono de candidatos a franquiciado

Este proyecto tiene como objetivo desarrollar un modelo predictivo que permita identificar candidatos con alta probabilidad de abandonar el proceso antes de la firma.

## Contenido

- Limpieza y preparación de datos
- Análisis exploratorio
- Segmentación de candidatos (clustering)
- Modelo predictivo (Regresión logística)
- Evaluación del modelo

## Tecnologías utilizadas

- Python
- Pandas
- Scikit-learn
- Matplotlib
